# NNNN - Short Decision Title

State the decision in 2-4 sentences. Write it as a rule a future contributor or
agent cannot easily misread.

Explain the forces behind the decision in 2-4 sentences. Name the obvious
alternative that was rejected and what would go wrong if it returned.

## Consequences

- State what kind of change now goes where because of this decision.
- Name any code path that must not be reintroduced so reviewers can grep for it.
